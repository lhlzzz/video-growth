# opening_3s_library

用途：建立开头 3 秒数据库，分析为什么停留、为什么划走，服务标题测试、结构优化、完播率研究和样片验证。

## 记录字段

- `hook_type`
- `attention_pattern`
- `emotion_trigger`
- `curiosity_gap`
- `trust_signal`

## 研究问题

1. 为什么停留？
2. 为什么划走？
3. 哪类开头更容易建立信任？
4. 哪类开头 AI 感更强？
5. 哪类开头平台风险更高？

## 禁止边界

- 不研究成交。
- 不研究收款。
- 不研究报价。
- 不研究转化。
- 不设计引流、私信、接单或自动销售链路。

## 文件

- `opening-template.md`：单条开头结构记录模板。
- `opening-log.md`：持续记录开头 3 秒结构。
