# Review Gate: 5 Checks

用途：发布前 5 项自查只作为审片 Gate；当前 B 模式下不授权发布。

## 适用对象

- 06/07 样片
- C5 备选低风险样片结构
- 后续任何本地样片、标题、脚本、分镜或字幕稿

## 5 项自查

| 检查项 | 通过标准 | 失败处理 |
| --- | --- | --- |
| AI 感检测 | 语言具体、有人味、场景清楚，不像纯 AI 流水线内容 | 禁止进入下一阶段，重写脚本和字幕 |
| 营销风险检测 | 不含保证、一定、最低价、包成交、稳赚、爆单等表达 | 禁止进入下一阶段，替换为研究/说明口径 |
| 承接感检测 | 不引导私信、企微、群、表单、付款、下单、接单或自动销售 | 禁止进入下一阶段，改为研究/审片/验证口径 |
| 搬运风险检测 | 不使用未授权原片、原声、封面、截图、评论或他人素材 | 禁止进入下一阶段，换原创或授权素材 |
| 内容标注检测 | 虚构、演绎、占位、待确认、待人工复核等标注清楚 | 禁止进入下一阶段，补标注后复审 |

## 结论模板

```text
样片编号：
检查日期：

1. AI 感检测：通过 / 失败
2. 营销风险检测：通过 / 失败
3. 承接感检测：通过 / 失败
4. 搬运风险检测：通过 / 失败
5. 内容标注检测：通过 / 失败

结论：通过 / 禁止进入下一阶段
原因：
下一步只允许动作：研究 / 审片 / 样片验证 / 结构优化 / 后台只读核验
```
