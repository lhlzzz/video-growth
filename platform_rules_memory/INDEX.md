# platform_rules_memory

用途：记录视频号平台规则变化和规则研究结论，只服务研究、审查、样片验证和内容结构优化。

## 分类

- AI 内容规则
- 短视频标注规则
- 营销内容规则
- 蓝 V 规则
- 虚构演绎规则
- 评论区规则
- 视频号治理公告

## 使用边界

- 只记录来自公开页面、平台公告、后台只读提示或人工确认的规则信息。
- 不凭空假设平台规则、账号权限、带货权限、收入权益或处罚机制。
- 需要浏览器资料时，使用项目隔离浏览器 MCP；失败或资料不足时标记 `BLOCKED: NEED_BROWSER_RESEARCH`。
- 不把规则研究直接升级为发布、引流、报价、收款、接单或自动销售动作。

## 文件

- `rules-log.md`：持续记录平台规则观察。
- `rule-template.md`：单条规则记录模板。
- `review-gate-5-checks.md`：发布前 5 项自查模板，仅作审片 Gate。
