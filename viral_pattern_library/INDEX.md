# viral_pattern_library

用途：记录视频号内容结构验证结果，只服务研究、审片、样片验证和内容结构优化，不服务发布、引流、报价、收款、接单或自动成交。

## 记录对象

- 标题结构
- 开头结构
- 视频时长
- 完播率
- 点赞率
- 评论率
- 收藏率
- 主页访问率

## 识别标签

- `高停留结构`
- `高信任结构`
- `高互动结构`
- `高关注结构`

## 使用边界

- 只记录公开可见数据、后台只读核验数据或内部样片验证数据。
- 不记录用户隐私、订单、付款、私信、联系方式或任何交易数据。
- ROI 研究仅限内容层，不计算真实成交或收款。
- 任何结论都必须标注样本来源和不确定性。

## 文件

- `pattern-log.md`：持续记录样片/公开视频结构与数据。
- `pattern-template.md`：单条结构分析模板。
- `sample-review-template.md`：06/07 样片深度审片模板。
- `sample-review-log.md`：06/07 样片审片记录。
